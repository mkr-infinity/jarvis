# J.A.R.V.I.S Desktop AI Assistant

A lightweight, Iron-Man inspired desktop AI assistant with voice capabilities.

## Features

- **JARVIS UI** - Iron-Man style arc reactor interface
- **Voice Mode** - Click mic for voice input with animated reactor
- **AI Providers** - Ollama (local), OpenAI, Anthropic, Gemini
- **Text-to-Speech** - JARVIS speaks responses
- **Chat History** - Projects and sessions
- **Keyboard** - Enter to send, ESC to close

## Quick Start

```bash
# Install
pip install -r requirements.txt
npm install

# Run
npm start
```

## Requirements

- Python 3.10+
- Node.js 18+
- Ollama (optional, for local AI)

## Setup

1. Run `npm start`
2. Enter API key in Settings (or use Ollama for free local AI)
3. Start chatting!

## Voice Commands

| Command | Action |
|---------|--------|
| `search [query]` | Search in browser |
| `open [url]` | Open URL |
| `screenshot` | Take screenshot |
| `volume 75` | Set volume |
